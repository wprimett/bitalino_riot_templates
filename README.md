### bitalino_riot_templates

A set of minimal code examples to receive data from the BITalino R-IoT using OSC

If there is a successful WiFi connection between the host computer and module, these scripts can be used to print out the incoming data.

The following assumes the standard network configuration of the R-IoT. If these have been changed, please change the IP adddress and port accordingly
